# Dashboard
Dashboard do Valor Bruto da Produção Agropecuária do Paraná (VBP)

O dashboard foi estruturado com cartões de indicadores, gráficos de barras, gráfico de rosca, tabela e segmentações de dados, permitindo análises por município, grupo de produção, cultura e núcleo regional. Entre os principais indicadores construídos destacam-se o Valor Bruto da Produção (VBP), o Top 10 Municípios por VBP, a Participação do VBP por Grupo de Produção, além da quantidade de municípios, culturas e núcleos regionais presentes na base
